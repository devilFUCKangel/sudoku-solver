Sudoku Champion Help

Introduction

Sudoku Champion is a powerful Sudoku program with a large number of features: You can create your own Sudoku settings, or have Sudoku Champion create one. You can save and load puzzles, and you can copy and paste them as text so you can easily share them by email. Sudoku Champion can solve a Sudoku on its own, or it can give you different forms of hints, or you can solve it all by yourself. Finally, you can also print Sudoku puzzles.

Quick Start

When you start Sudoku Champion, it opens with a newly generated puzzle for you to solve. Every puzzle has a unique solution. Sudoku Champion always highlights the currently active cell by drawing its border in red. To change the value of the highlighted cell, just type a number from 1 to 9 on the keyboard. Typing 0, delete or backspace will clear the current value of the cell. You can move the active cell with the arrow keys or by clicking a different cell with the mouse. While you solve the puzzle, Sudoku Champion offers you different types of help:
  • Cells which can be solved can be highlighted
  • The highlighting can be coded according to the reason for which the cell is solvable
  • Right-clicking a cell with the mouse will display the remaining possible values of that cell in the status bar
  • You can also display all possible remaining numbers for a cell inside the cell as "pencilmarks".
  • By double-clicking on a cell which can be resolved through simple scanning, Sudoku Champion will fill in the cell's value for you
  • Sudoku Champion can count the number of solutions a given Sudoku has
  • Sudoku Champion can solve the entire Sudoku for you

Menu Command Reference

The File Menu

  • New
    creates a new Sudoku with a unique solution
  • New Empty
    creates an empty Sudoku
  • Open
    loads a Sudoku from a file
  • Save
    saves a Sudoku to a file
  • Print
    prints the current Sudoku in the current style, and 5 more Sudokus to go with it - to check this...
  • Print Preview
    ...lets you see what you are going to print
  • Exit
    terminates the program

The Edit Menu

  • Undo
    takes back the last change you made to the Sudoku
  • Redo
    redoes the last change you undid
  • Copy
    copies the Sudoku as a string to the clipboard. You can send this to a friend in an email, and she can...
  • Paste
    ...paste it into her copy of Sudoku Champion

The Tools Menu

  • Show remaining possibilities
    shows all remaining possibilities as small numbers inside the cell (pencilmarks).
  • Highlight resolvable cells
    highlights cells which can be solved in gold
  • Show Hint for Resolvable Cells
    gives you a hint why the cell can be solved. There are 4 different reasons why a cell can be solved: If there is only one remaining value a cell can take, or if the cell is the only cell in a row/column/region which can take a certain value. The image on the right shows all of these hints: a small box in the lower right corner means that this cell can only take one value. A larger box in the cell center means that this cell is the only cell in this region which can take a specific value. A vertical bar in a cell means that this cell is the only cell in a column which can take a specific value. A horizontal bar means that this is the only cell in a row which can take a specific value.
  • Count Solutions
    counts the number of solutions the Sudoku has. A well-formed Sudoku has a single solution. If the number of solutions exceeds 100, Sudoku Champion stops searching for further solutions.
  • Solve Sudoku
    solves the Sudoku. You can use Undo and Redo to step through the solution process

The Help Menu

  • Contents
    displays this help file
  • Sudoku Champion Homepage
    goes to the Sudoku Champion homepage on the internet, where you can find the latest version
  • About
    displays an about box

Miscellania

I wrote Sudoku Champion as an excercise since I was learning C#. The best way to learn a programming language is to actually write a program and learn by doing. I got the idea to write a Sudoku program from Léonie Fierz (who is no relation of mine): She was a student in a Science course for Swiss youths, and told me that she was supposed to write a Sudoku program as an assignment in class. I was motivated to write Sudoku Champion when I saw the most beautiful woman of the clinic Königsfelden solve one on the train from Brugg to Zürich. I finally started writing it in march 2006 in my mountain retreat in the Engadin.
